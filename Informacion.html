El Formulario Pasajero tiene 5 funciones, las cuales facilitan a los usuarios poder ver, seleccionar, insertar, actualizar y eliminar

con estas opciones se tiene una mejor funcionalidad del programa y control en la base de datos y APIS


N# Formulario Pasajero
1 GeTAll Este parte del Formulario sirve para poder mostrar los datos de estan en las APIS
2 GetOne Este metodo mostrar un registro seleccionado por el usuario, listo para poder editar
3 GetInsert Este metodo sirva para poder Insertar un nuevo registro los cuales son insertados directamnete a la base de datos
4 GetUpdate Este metodo sirve para actualizar un registro seleccionado con el metodo GetOne
5 GetDelete Este metodo sirve para poder eliminar un registro
